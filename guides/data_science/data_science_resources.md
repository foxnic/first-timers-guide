# Data Science Resources

If you're interested in learning about data science it can be difficult to know where to start and to find good resources. Below is a list of useful resources to help get you started.


## Books

- [Doing Data Science](https://www.oreilly.com/library/view/doing-data-science/9781449363871/)
- [Statistics Explained](https://www.amazon.co.uk/Statistics-Explained-Introductory-Guide-Scientists/dp/0521183286)
- [Introduction to Machine Learning with Python](https://www.oreilly.com/library/view/introduction-to-machine/9781449369880/)
- [R For Data Science](https://r4ds.had.co.nz/)
- [Applied Predictive Modelling](https://www.springer.com/gp/book/9781461468486)
- [Python Data Science Handbook](https://jakevdp.github.io/PythonDataScienceHandbook/)
- [The Art of Statistics](https://www.amazon.co.uk/Art-Statistics-Learning-Pelican-Books/dp/0241398630/)
- [Practical Statistics for Data Scientists](https://www.oreilly.com/library/view/practical-statistics-for/9781491952955/)


## Communities

- [R For Data Science Community](https://www.rfordatasci.com/)
- [Data Visualisation Society](https://www.datavisualizationsociety.com/)


## Free Courses

- [Khan Academy - Various Maths Tutorials](https://www.khanacademy.org/math)
- [Python Fundamentals For Data Science](https://app.dataquest.io/login?target-url=%2Fcourse%2Fpython-for-data-science-fundamentals)
-[Introduction to Shell for data Science](https://www.datacamp.com/courses/introduction-to-shell-for-data-science)
